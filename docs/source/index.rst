scikit-duplo: Simple Blocks for Scikit-Learn
===============================================

``scikit-duplo`` is a python library for extending the functionality of
scikit learn machine learning pipelines. It was inspired by and borrows
from the scikit-lego library (which is recommended). We have simply designed
several classes that implement simple machine learning ideas in a reusable
format.
